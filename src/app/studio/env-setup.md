# Настройка окружения для KARTO Studio

## Необходимые API ключи

### 1. Replicate API (обязательно)
Используется для: удаление фона, улучшение качества, генерация фонов

1. Зайдите на https://replicate.com/
2. Создайте аккаунт
3. Перейдите в https://replicate.com/account/api-tokens
4. Скопируйте токен

Создайте файл `.env.local` в корне проекта:
```
REPLICATE_API_TOKEN=r8_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### Стоимость на Replicate:
- Удаление фона (rembg): ~$0.01-0.02 за запрос
- Улучшение качества (Real-ESRGAN): ~$0.02-0.03 за запрос
- Генерация фона (SDXL): ~$0.03-0.05 за запрос
- Распознавание товара (LLaVA): ~$0.01-0.02 за запрос

**Итого на 1 карточку: ~$0.07-0.12 (5-10 рублей)**

## Альтернативы для России

Если Replicate недоступен:

### Вариант 1: Прокси-сервер
Арендуйте VPS в Европе (€5-10/мес) и настройте прокси

### Вариант 2: Российские API
- **Kandinsky** (Сбер) - генерация изображений
- **YandexART** - генерация изображений
- **GigaChat** / **YandexGPT** - генерация текстов

## Запуск

```bash
# 1. Создайте .env.local с ключами
# 2. Запустите dev сервер
npm run dev

# 3. Откройте http://localhost:3000/studio
```
