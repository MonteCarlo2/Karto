# SEO workspace: karto.pro

Локальная папка контекста для OpenSEO Agent Skills и Cursor. Агент читает её при `/seo-project-setup`, `/keyword-research` и т.д.

## Сайт

| Поле | Значение |
|------|----------|
| Домен | https://karto.pro |
| Язык | Русский (RU) |
| Рынок | Россия, селлеры WB / Ozon / Яндекс Маркет |
| CMS | Next.js (этот репозиторий) |

## Продукт (кратко)

**KARTO** — ИИ-помощник продавца маркетплейсов:

- **Поток** — карточки, SEO-описания, визуал, цена за один цикл
- **Свободное творчество** — фото и видео по промптам
- **Автоответы** — ответы на отзывы WB, Ozon, Яндекс Маркет (полуавто и авто)
- **Юнит-экономика** — калькулятор маржи WB/Ozon

## Цели SEO (6–12 месяцев)

1. Органические регистрации селлеров (не только брендовые запросы «karto»).
2. Топ-10 по коммерческим запросам: автоответы, генерация карточек, ИИ для маркетплейсов.
3. Рост кликов из Google Search Console на `/`, `/pricing`, лендинг автоответов.

## Метрики успеха

- Небрендовый organic traffic +30% квартал к кварталу
- 15+ целевых ключей в топ-20 Google (RU)
- CTR главной > 3% по money-запросам

## Приоритетные страницы

| URL | Зачем |
|-----|--------|
| `/` | Главный лендинг, highest priority |
| `/pricing` | Коммерческие запросы, тарифы |
| `/studio/auto-replies` | Воронка автоответов |
| `/login` | Низкий приоритет, но индексируется |

Юридические: `/privacy`, `/terms`, `/payments-policy` — поддержка доверия, не лидоген.

## Конкуренты (для competitive-landscape / competitor-analysis)

- Ответо (автоответы)
- MPStats / Moneyplace (аналитика, не 1:1)
- Конструкторы карточек / фриланс
- Встроенные ИИ-инструменты маркетплейсов

## Позиционирование

- Дешевле Ответо на входе, единая платформа (контент + отзывы)
- Не «ещё один MPStats» — фокус на **создании контента и ответах**, не на аналитике ниши
- Для селлера с оборотом, которому нужен результат «сегодня», а не отчёт на 40 страниц

## OpenSEO project

1. Создайте проект **karto.pro** в [openseo.so](https://openseo.so) или локальном Docker.
2. Подключите **Google Search Console** для `karto.pro`.
3. В Cursor: MCP `openseo` → `https://app.openseo.so/mcp` (логин при первом подключении).

## Папки

```text
seo-workspace/karto-pro/
  README.md          ← этот файл
  keywords/          ← экспорт из OpenSEO / GSC CSV
  competitors/       ← заметки по конкурентам
  content/           ← брифы статей, FAQ, мета-тексты
  reports/           ← еженедельные SEO-отчёты
  gsc/               ← CSV из Search Console (если без OAuth)
```

## Рекомендуемый цикл (еженедельно)

1. **GSC** — какие запросы уже дают показы (`get_search_console_performance` через MCP).
2. **keyword-research** — новые кластеры под `/pricing` и автоответы.
3. **rank tracking** — 30–50 ключей в OpenSEO.
4. **Правки в коде** — title/description, FAQ, sitemap (`src/app/sitemap.ts`).
5. **site audit** — раз в месяц, чинить критичное.
